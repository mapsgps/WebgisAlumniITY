Web ini hanya semata untuk mengetahui persebaran Alumni INSTITUT TEKNOLOGI YOGYAKARTA dengan format Spasial
Mohon diisi klik TAMBAH DATA dan siapkan Lokasi Koordinat Rumah/Instansi Anda dengan Format Desimal. Tulis koordinat dengan: Diawali TANDA PETIK (') dan dipisah TANDA TITIK (.)

- LU/LS: Lintang Utara contoh : '2.12345 dan Lintang Selatan setelah tanda petik diberi tanda min (-) contoh: '-7.12345
- Bujur (BT): Contoh: '110.12345

Apabila kesulitan lihat di web kemudian arahkan/zoom-in lokasi rumah/instansi Anda, klik akan muncul Koordinat Desimal, catat/salin masukan di Form

Untuk menjaga privacy: Data alamat rumah, no.telp dan email tidak akan ditampilkan dalam webgis
